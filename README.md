# Centro Deportivo

Simple CLI application in Python that calculates the price a client must pay in a sports center based on membership status and age.

This project is part of my programming learning portfolio and demonstrates modular program structure.

---

## Features

- Ask if the user is a member
- Search members by ID
- Handle invalid input
- Calculate final price based on membership status and age

---

## Project Structure

centro_deportivo/

src/  
├── main.py  
├── ui.py  
├── services.py  
└── data.py  

- **main.py** → program flow
- **ui.py** → user interaction
- **services.py** → business logic
- **data.py** → system data

---

## How to Run

From the project root run:

python src/main.py
